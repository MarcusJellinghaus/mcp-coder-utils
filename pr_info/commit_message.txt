feat: add subprocess_streaming module with inactivity watchdog

Add subprocess_streaming.py with StreamResult iterator wrapper and
stream_subprocess generator for real-time stdout line streaming.
Includes an inactivity watchdog that kills hung processes after a
configurable timeout.

- StreamResult: iterator with .result property for final CommandResult
- stream_subprocess: generator yielding stdout lines with optional
  inactivity watchdog using a background thread
- Platform-aware process killing (taskkill on Windows, killpg on Unix)
- 3 unit tests covering basic streaming, active process no-timeout,
  and inactivity timeout kill scenarios
